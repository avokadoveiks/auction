Place your local bot avatar photos here:

- humans/: h1.jpg … h10.jpg (or .png/.jpeg)
- animals/: a1.jpg … a10.jpg
- landscapes/: l1.jpg … l10.jpg

The game will try humans first, then animals, then landscapes per bot index, and fall back to SVG icons in assets/bots/.

Included in this repo: lightweight SVG placeholders for each category (h1–h10, a1–a10, l1–l10). You can replace any of them with real local photos — just keep the same filenames. JPG/PNG/JPEG/SVG are supported.

Tips:
- Use square images (e.g., 512×512) for best fit.
- Keep file sizes small (<200KB) for faster loads on mobile.
- If a file is missing, the game will gracefully fall back to the built-in botX.svg icons.
