Place your desktop wallpaper image here as wallpaper.jpg.

Accepted formats: .jpg, .jpeg, .png (update index.html preload if not .jpg).

Alternative: pass a URL param to index.html, for example:

  index.html?wallpaper=https://example.com/your-image.jpg

Security note: if you use a remote URL, your browser may block mixed-content or cross-origin images in some contexts.
