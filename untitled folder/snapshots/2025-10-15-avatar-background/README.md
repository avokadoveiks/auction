## Ключевые изменения

### CSS
- Аватар на фоне через `::before` на всю колонку до кнопок (`bottom: 80px`)
- `.column.has-bet::before` - opacity 0.35 для хорошей видимости
- `.bet-info` - фиксированная высота 24px, чтобы колонки не прыгали
- `.bet-avatar` и `.bet-extra` - полностью скрыты
- `.column` - `max-width: 20%` для фиксированной ширины

### JavaScript
- `updateTimerDisplay()` - формат M:SS вместо MM
- `showBetInfo()` - устанавливает `data-bg-emoji` и класс `has-bet`
- `resetColumn()` - очищает фон аватара
- `showWinnerNotification()` - отключена (return)

### HTML
- Таймер перенесён из `.column-body` в отдельный блок перед `.bet-controls`
- Убран дублированный блок winner-notification

## Проблемы решены
