# АВОКАДО1 - Версия с аватаром на фоне

## Быстрое восстановление
Когда скажете "восстанови авокадо1", я выполню:
```bash
cp snapshots/avokado1/v1.html ./v1.html
cp snapshots/avokado1/v1.css ./v1.css
cp snapshots/avokado1/v1.js ./v1.js
```

## Что в этой версии
✅ Аватар на фоне колонки при ставке (35% прозрачность)
✅ Таймер в формате M:SS (0:23, 1:05)
✅ Таймер внизу колонки над кнопками
✅ Без попапа победителя
✅ Только имя игрока (без маленькой аватарки)
✅ Колонки НЕ увеличиваются при ставке
✅ iPhone landscape - все колонки без прокрутки
